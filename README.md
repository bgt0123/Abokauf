# Abokauf
Dies ist das Repository für die Abokauf-React (Frontend) Prüfung. Die Applikation wurde im Google Chrome entwickelt und eine Prüfung von weiteren Browsern ist nicht Teil der Arbeit gewesen.
Aufgabe und ähnliche nützliche Dokumente sind in ./documentation zu finden. Das Backend wird im /api Ordner abgelegt, ist aber nicht teil dieser Arbeit.

## Frameworks, Programmiersprachen & Tools
- React
- Typescript
- Redux

## Styling:
- Icons: Lucide for React (https://lucide.dev/guide/react/) -> import { CheckCircle, Home } from 'lucide-react';
- Tailwind: https://tailwindcss.com/

## App Start
```npm install```

```npm run dev```
