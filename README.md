# Abokauf
Dies ist das Repository für die Abokauf-React (Frontend) Prüfung. Die Applikation wurde im Google Chrome entwickelt und eine Prüfung von weiteren Browsern ist nicht Teil der Arbeit gewesen.
Aufgabe und ähnliche nützliche Dokumente sind in ./documentation zu finden.

# Frameworks, Programmiersprachen & Tools
- React
- Typescript
- Redux
- Redux Dev Tools (Google Chrome Extension)

# App Start
```npm install```

```npm run dev```

# Frontend
- Es wird ein React Router genutzt für die Navigation über mehrere Seiten hinweg.

# Backend
- Das Backend wird im /api Ordner abgelegt, ist aber nicht teil dieser Arbeit.
